# Food Delivery Analytics Dashboard - Power BI

## Project Overview

This project analyzes food delivery business data using Power BI to uncover insights related to sales performance, customer behavior, delivery efficiency, and business growth.

## Dashboard Insights

- Revenue Analysis
- Order Trends
- Customer Analysis
- Delivery Performance
- Restaurant Performance
- City-wise Analysis
- KPI Tracking

## Tools Used

- Power BI
- DAX
- Power Query
- Data Modeling

## Dashboard Features

- Interactive Slicers
- KPI Cards
- Drill-Down Analysis
- Conditional Formatting
- Dynamic Visualizations

## Key Business Insights

- Identified peak ordering periods.
- Analyzed top-performing restaurants.
- Monitored delivery efficiency and delays.
- Evaluated customer ordering patterns.
- Tracked revenue growth and order volume.
